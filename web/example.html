Website Integration

Quick Integration

Add Moses's AI recruitment assistant to your website in 3 simple steps:

1. Include the JavaScript client:

<script src="https://cdn.jsdelivr.net/gh/moses-omondi/ai-recruitment-assistant@main/web/moses-ai-client.js"></script>

2. Add a container element:

<div id="moses-ai-chat"></div>

3. Initialize the chat widget:

<script> document.addEventListener('DOMContentLoaded', function() { new MosesAIChatWidget({ container: 'moses-ai-chat', apiUrl: 'https://your-api-endpoint.com', theme: 'professional', welcomeMessage: "Hi! Ask me about Moses's DevSecOps, MLOps, and AI security expertise!" }); }); </script>

Advanced Usage

For custom implementations, use the JavaScript client directly:

const client = new MosesAIClient({ apiUrl: 'https://your-api-endpoint.com', debug: true, onMessageStart: (message) => console.log('Sending:', message), onMessageComplete: (response) => console.log('Received:', response), onError: (error) => console.error('Error:', error) }); // Send a message const response = await client.chat("Tell me about Moses's DevSecOps experience"); console.log(response.response); // Get capabilities const capabilities = await client.getCapabilities(); console.log(capabilities.expertise_areas);

API Endpoints

  • POST /chat - Chat with Moses's AI assistant
  • GET /capabilities - Get expertise areas and role types
  • GET /sample-questions - Get sample recruitment questions
